## 서론
MySQL을 사용하다보면 MySQL의 프롬프트를 변경해야할 경우가 생기는데 이 때 다음과 MySQL 콘솔에서 다음과 같이 입력하면 된다.
<!--more-->

```
prompt {프롬프트에 나타내고 싶은 문자열}
```

```
prompt [saltfactory-mysql]
```
